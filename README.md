# Pictionary
Pictionary for Small Basic

## Usage
1. Compile all .sb files. (except maker folder)
1. Run Pictionary.exe .
1. Click keyword in the program.

## System Requirements
- Windows Vista or later
- .NET Framework 4.5 or later
- LitDev Extension 1.2 or later
